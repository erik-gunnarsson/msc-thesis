\noindent\textit{Notes.}

\noindent\textbf{Panel A.}
Outcome is $\ln(\text{H\_EMPE})$, the log of hours worked.
Figures in parentheses are standard errors clustered by country.
Stars are omitted from robot-related terms (see Panel~B).

\noindent\textbf{Panel B.}
Each column shows $p$-values for that column's main robot-related term (see the ``Focal term'' row).
$p_{\text{cluster}}$ uses standard cluster-robust SEs by country.
$p_{\text{wild}}$ is a wild cluster bootstrap (Webb, 2023); $B$ is the number of bootstrap draws, and $B = 999$ is the headline row.
No robot term clears $p < 0.10$ on the wild bootstrap, so stars are omitted.
In column~(2), the coordination interaction is below 5\% using clustering ($p = 0.020$) but not using the wild bootstrap ($p = 0.108$), which often happens when there are few clusters ($G = 25$ countries).
Column~(5) follows Eq.~2b: the three-way term is an extra diagnostic on top of the coordination term.

\noindent\textbf{Panel C.}
Country counts vary across columns because institutional data availability differs by moderator.
Column~(3) uses a restricted 15-country sample due to limited adjusted coverage data.
Institutional moderators are frozen at 1990--1995 baseline levels.
Robot intensity is scaled using employment from before the sample period.
